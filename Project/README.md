# 🌙 NightPick - ค้นหาบาร์สุดฮิปในกรุงเทพฯ (Bangkok Bar Discovery App)

## 📝 เกี่ยวกับโปรเจกต์นี้
โปรเจกต์ **NightPick** เป็นส่วนหนึ่งของวิชา **CP213 การพัฒนาแอปพลิเคชันบนอุปกรณ์เคลื่อนที่** โดยแนวคิดของแอปนี้เกิดจากการที่ผมต้องการสร้างเครื่องมือที่ช่วยให้การหาที่เที่ยวหรือบาร์ในกรุงเทพฯ เป็นเรื่องง่ายและสนุกขึ้น ไม่ใช่แค่การดูรีวิวทั่วไป แต่ต้องการให้ข้อมูลที่ "สดใหม่" และ "ใช้งานได้จริง" เช่น ร้านเปิดอยู่ไหม หรืออยู่ห่างจากเรากี่กิโลเมตรครับ

---

## ✨ ฟีเจอร์ที่ผมพัฒนาในแอปนี้

ผมตั้งใจออกแบบแอปให้มีฟังก์ชันที่ครบถ้วนสำหรับการใช้งานจริง โดยแบ่งออกเป็นส่วนหลักๆ ดังนี้ครับ:

### 1. หน้าเปิดตัวแอป (Splash Screen)
*   ทำหน้าจอเปิดแอป (Splash Screen) พร้อมโลโก้ร้าน NightPick ที่ผมออกแบบเอง โดยใช้ระบบหน่วงเวลา 2 วินาทีก่อนจะพาเข้าสู่หน้าหลักโดยอัตโนมัติ เพื่อความพรีเมียมครับ

### 2. ระบบตำแหน่งและระยะทาง (Real-time Location)
*   **คำนวณระยะทางอัตโนมัติ:** แอปจะขอสิทธิ์เข้าถึง GPS และคำนวณระยะทางจาก "จุดที่ผมอยู่" ไปยังร้านแต่ละแห่งแบบเรียลไทม์ (ถ้าขยับตัวเลขกิโลจะเปลี่ยนตามครับ)
*   **ระบบนำทาง:** ผมเพิ่มปุ่มนำทางที่จะส่งพิกัดร้านไปยัง Google Maps ให้ผู้ใช้สามารถกดเดินหรือขับรถไปที่ร้านได้ทันที

### 3. ฟีเจอร์ "คืนนี้ไปไหนดี?" (Surprise Me / Randomizer)
*   เป็นฟีเจอร์ที่ผมเพิ่มเข้ามาสำหรับคนที่เลือกไม่ได้ว่าจะไปร้านไหนดีครับ เมื่อกดปุ่มสุ่มสีทองที่หน้าแรก แอปจะเลือกบาร์ที่น่าสนใจมาให้ 1 ร้าน และพาไปดูรายละเอียดทันที เป็นลูกเล่นที่เข้ากับสายปาร์ตี้ครับ

### 4. ระบบสถานะร้านแบบเรียลไทม์ (Live Opening Status)
*   ผมเขียนระบบให้แอปเช็คเวลาปัจจุบันของเครื่อง แล้วนำไปเปรียบเทียบกับเวลาเปิด-ปิดของร้าน เพื่อบอกว่าตอนนี้ร้าน **"เปิดอยู่"**, **"ปิดแล้ว"** หรือ **"ใกล้จะปิดแล้ว"** ครับ

### 5. การค้นหาและกรองข้อมูล (Search & Categories)
*   สามารถค้นหาชื่อร้านได้จากช่องค้นหา (Instant Search) และมีการกรองตามหมวดหมู่บาร์ เช่น Rooftop, Jazz, Cocktail เพื่อให้หาแนวที่ชอบได้ง่ายขึ้นครับ

### 6. หน้ารายละเอียดร้าน (Detailed View)
*   แสดงแกลเลอรีรูปภาพแบบเลื่อนได้ (Image Pager)
*   บอกข้อมูลเมนูแนะนำ (Signature Drinks) พร้อมราคา
*   มีปุ่ม "โทรจอง" ที่กดแล้วจะดึงเบอร์ร้านเข้าหน้าโทรศัพท์ให้เลยครับ

### 7. ระบบร้านโปรดและแชร์ข้อมูล (Favorites & Sharing)
*   **Favorites:** สามารถกดหัวใจเพื่อบันทึกร้านที่ชอบไว้ดูภายหลังได้ ข้อมูลจะถูกบันทึกไว้ในเครื่อง (แม้ปิดแอปไปข้อมูลก็ไม่หายครับ)
*   **Sharing:** ผมเพิ่มปุ่มแชร์ร้านให้เพื่อน เพื่อส่งข้อมูลชื่อร้าน ที่อยู่ และเบอร์โทรผ่านแอปแชทต่างๆ ได้ง่ายๆ ครับ

### 8. ระบบสลับธีม (Dark & Light Mode)
*   ผมได้เขียนระบบให้แอปสามารถสลับระหว่าง **โหมดมืด (Dark Mode)** สำหรับใช้กลางคืน และ **โหมดสว่าง (Light Mode)** สำหรับใช้กลางวัน โดยสามารถเลือกสลับได้ในหน้าโปรไฟล์ครับ

## เทคนิคที่ใช้ในการพัฒนา (Technical Detail)

ในการทำโปรเจกต์นี้ ผมเลือกใช้เทคโนโลยีล่าสุดของ Android เพื่อให้แอปมีความลื่นไหลและทันสมัยที่สุดครับ:
*   **Kotlin & Jetpack Compose:** ผมใช้ Compose ในการเขียน UI ทั้งหมด 100% ทำให้จัดการหน้าจอได้ง่ายและสวยงาม
*   **MVVM Architecture:** ผมแยกส่วนของ Logic (ViewModel) ออกจาก UI อย่างชัดเจน เพื่อให้โค้ดเป็นระเบียบและไม่ซับซ้อนครับ
*   **State Management:** ใช้ **StateFlow** ในการจัดการสถานะข้อมูล (เช่น ข้อมูลร้าน, ระยะทาง) เพื่อให้ UI อัปเดตทันทีเมื่อข้อมูลเปลี่ยน
*   **Data Persistence:** ใช้ **SharedPreferences** ในการเก็บค่าการตั้งค่าต่างๆ ของผู้ใช้ เช่น ชื่อ-อีเมล, ธีมที่เลือก และลิสต์ร้านโปรด
*   **Image Loading:** ใช้ Library ที่ชื่อว่า **Coil** ในการดึงรูปภาพสวยๆ จาก URL มาแสดงผลครับ

## โครงสร้างโฟลเดอร์ในโปรเจกต์
ผมจัดเก็บไฟล์แยกตามหน้าที่ (Package by Layer/Feature) เพื่อให้อาจารย์ตรวจได้ง่ายครับ:
*   `data/`: เก็บข้อมูลจำลอง (Mock Data) และตัวจัดการไฟล์ (Managers)
*   `ui/screens/`: รวบรวมหน้าจอหลักทั้งหมด (Home, Detail, Favorites, Profile, Splash)
*   `ui/theme/`: ส่วนกำหนดสี ฟอนต์ และการสลับธีม
*   `utils/`: ฟังก์ชันเสริมที่ผมเขียนขึ้นเอง เช่น การคำนวณกิโลเมตรจาก GPS
